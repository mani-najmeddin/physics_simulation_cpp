# Quantum Gravity Physics Engine

A C++ physics engine that computes six quantum gravity and general relativity
formulas, exports results as CSV, and visualizes them with Python/matplotlib.

---

## Files

```
quantum_gravity.cpp   — physics engine (all computation)
plot.py               — visualizer (reads CSV, draws plots)
README.md             — this file
output/               — generated by the engine at runtime
```

---

## How to Run

**Step 1 — Compile and run the engine:**
```bash
g++ -O2 -o engine quantum_gravity.cpp
./engine
```
This prints results to the terminal and writes CSV files into `output/`.

**Step 2 — Plot the results:**
```bash
python3 plot.py
```
This reads the CSVs and saves `output/quantum_gravity_plot.png`.

## What the Engine Computes

### 1. Feynman Path Integral
```
K(xf,tf ; xi,ti) = ∫ D[x(t)] exp(iS[x]/ħ)
```
Monte Carlo method — samples 50,000 random paths between two points,
computes the action S for each, and sums the phase contributions to
approximate the quantum propagator K.

Output: `output/path_integral.csv`
Columns: `path_id, is_classical, t0..tN, action`

---

### 2. Hawking Radiation
```
T_H = ħc³ / (8πGMk_B)
dM/dt = -ħc⁴ / (15360π G² M²)
```
Computes Hawking temperature, luminosity, Bekenstein-Hawking entropy,
and evaporation time across a log-spaced range of black hole masses.

Output: `output/hawking_mass_sweep.csv`, `output/hawking_evap.csv`
Columns: `mass_solar, temperature_K, radius_m, luminosity_W, evap_time_s, entropy`

---

### 3. Einstein Field Equations
```
G_μν + Λg_μν = (8πG/c⁴) T_μν
```
Evaluates the Schwarzschild metric tensor components and the Kretschner
curvature scalar K = 48G²M²/(c⁴r⁶) as a function of radius.

Output: `output/einstein_metric.csv`
Columns: `r_over_rs, g_tt, g_rr, g_thth, kretschner_K, redshift_z, time_dilation`

---

### 4. Geodesic Equations
```
d²xᵘ/dτ² + Γᵘ_αβ (dxᵅ/dτ)(dx^β/dτ) = 0
```
Numerically integrates a test particle's trajectory in Schwarzschild
spacetime using the Christoffel symbols. Also computes the effective
potential for both GR and Newtonian gravity.

Output: `output/geodesic_orbit.csv`, `output/geodesic_veff.csv`
Columns: `step, r_m, r_over_rs, phi_rad, x_m, y_m` / `r_over_rs, V_eff_GR, V_eff_Newton`

---

### 5. Schrödinger Equation
```
iħ ∂ψ/∂t = [-ħ²/2m ∇² + V(x)] ψ
```
Time-evolves a Gaussian wavepacket in a harmonic oscillator potential
using the Crank-Nicolson method (unitary, norm-preserving).
Snapshots of |ψ|² are saved at t = 0, 40, 100, and 200 steps.

Output: `output/schrodinger_wave.csv`
Columns: `x, prob_t0, prob_t40, prob_t100, prob_t200, potential`

---

### 6. Schwarzschild Metric
```
ds² = -(1-rₛ/r)c²dt² + (1-rₛ/r)⁻¹dr² + r²dΩ²
```
Computes gravitational redshift z, time dilation dτ/dt, and light
deflection angle as a function of radius around a black hole.

Output: `output/schwarzschild_effects.csv`
Columns: `r_over_rs, redshift_z, time_dilation, light_deflect_deg, g_tt, g_rr`

## What the Plot Shows

| Panel | Source CSV | What you see |
|-------|-----------|--------------|
| 01 | path_integral.csv | 80 sampled paths + classical path + phase amplitude |
| 02 | hawking_mass_sweep.csv | T_H and evaporation time vs mass |
| 03 | geodesic_veff.csv + geodesic_orbit.csv | Effective potential + precessing orbit |
| 04 | schrodinger_wave.csv | Wavepacket evolution through harmonic potential |
| 05 | einstein_metric.csv | Flamm paraboloid embedding + Kretschner scalar |
| 06 | schwarzschild_effects.csv | Redshift, time dilation, light deflection vs r |

## Reference Black Hole

All single-BH calculations use **5 solar masses** (M = 5 × 1.989×10³⁰ kg):

| Property | Value |
|----------|-------|
| Schwarzschild radius | ~14.8 km |
| Hawking temperature | ~1.23×10⁻⁸ K |
| ISCO radius | ~44.3 km |
| Photon sphere | ~22.2 km |
| Evaporation time | ~8.27×10⁷⁶ years |
